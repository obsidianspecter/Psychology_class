Week 6: Memory

Introduction to Memory

Memory is the mental process of encoding, storing, and retrieving information. It is essential for learning and influences our ability to function in daily life. Understanding how memory works can help improve learning strategies and address memory-related issues.

Models of Memory

1. Multi-Store Model of Memory

The multi-store model, proposed by Atkinson and Shiffrin, describes memory as consisting of three distinct stores:

  • Sensory Memory:
    • Duration: Very brief (milliseconds to seconds).
    • Capacity: Large but fleeting; holds sensory information just long enough for it to be processed.
    • Example: The brief retention of visual images after looking away from a scene.
  • Short-Term Memory (STM):
    • Duration: About 15-30 seconds without rehearsal.
    • Capacity: Limited (7 ± 2 items, as proposed by Miller).
    • Function: Holds information temporarily for processing and manipulation.
    • Example: Remembering a phone number long enough to dial it.
  • Long-Term Memory (LTM):
    • Duration: Potentially unlimited; can last from minutes to a lifetime.
    • Capacity: Vast; stores information that has been encoded through rehearsal or meaningful association.
    • Example: Recalling your first day at school years later.

2. Working Memory Model

Proposed by Baddeley and Hitch, the working memory model expands on short-term memory by describing it as an active processing system with multiple components:

  • Central Executive: The control center that directs attention and coordinates activities among the subsystems.
  • Phonological Loop: Processes auditory information and verbal material.
  • Visuospatial Sketchpad: Handles visual and spatial information.
  • Episodic Buffer: Integrates information from different sources into a coherent episode.

Types of Memory

1. Explicit (Declarative) Memory

  • Definition: Memory that requires conscious thought; includes facts and events.
  • Subtypes:
    • Episodic Memory: Personal experiences and specific events (e.g., your last birthday party).
    • Semantic Memory: General knowledge and facts about the world (e.g., Paris is the capital of France).

2. Implicit (Non-declarative) Memory

  • Definition: Memory that does not require conscious thought; involves skills and conditioned responses.
  • Subtypes:
    • Procedural Memory: Skills and actions (e.g., riding a bike).
    • Priming: Exposure to one stimulus influences response to another stimulus (e.g., hearing the word "yellow" may make you recognize the word "banana" faster).

Factors Affecting Memory Retention and Recall

1. Encoding Strategies

  • Rehearsal: Repeating information to enhance retention (e.g., studying flashcards).
  • Elaborative Rehearsal: Connecting new information with existing knowledge for deeper processing (e.g., creating a story with new vocabulary words).

2. Contextual Cues

  • The environment in which learning occurs can serve as a cue for retrieval. For example, studying in the same location where you will take an exam can improve recall.

3. Emotional State

  • Emotions can enhance or impair memory retention. Strong emotional experiences are often remembered more vividly due to the involvement of the amygdala in processing emotions.

4. Interference

  • Proactive Interference: Older memories interfere with the retrieval of newer memories (e.g., difficulty remembering a new phone number because of an old one).
  • Retroactive Interference: New memories disrupt the retrieval of older memories (e.g., learning a new language may interfere with recalling vocabulary from a previous language learned).

Applications in Social Psychology

Understanding memory is crucial in social psychology:

  • How eyewitness testimonies can be influenced by memory distortions or biases.
  • The role of collective memory in shaping group identity and social narratives.

Applications in Industrial Psychology

In organizational settings:

  • Training programs can be designed to enhance memory retention through effective encoding strategies.
  • Understanding how stress affects memory can inform employee wellness initiatives.

Activities

  • Memory Experiment: Conduct a simple class experiment where students memorize a list of words using different encoding strategies (e.g., rote rehearsal vs. elaborative rehearsal) and compare recall rates.
  • Group Discussion: Have students share personal anecdotes about how context or emotion influenced their recollection of an event.
